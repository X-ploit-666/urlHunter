# URLHunter

Automated URL Discovery & XSS Recon Pipeline

URLHunter automates:

- Historical URL discovery
- URL collection
- Scope filtering
- Live endpoint verification
- JavaScript crawling
- XSS parameter discovery
- Dalfox scanning

## Features

✓ waybackurls integration
✓ gau integration
✓ katana crawling
✓ gf parameter filtering
✓ kxss/Gxss reflection analysis
✓ Dalfox automation

## Installation

git clone https://github.com/X-ploit-666/URLHunter.git

cd URLHunter

chmod +x urlhunter

./urlhunter domains.txt

## Requirements

- waybackurls
- gau
- httpx
- katana
- gf
- uro
- Gxss
